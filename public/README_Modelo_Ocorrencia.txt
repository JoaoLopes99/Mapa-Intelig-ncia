MODELO DE OCORRÊNCIA - INSTRUÇÕES DE USO

Este arquivo Excel (.xlsx) contém um modelo para cadastro de ocorrências no sistema.

COLUNAS DO MODELO:
- Tipo de Ocorrência: Furto, Roubo, Homicídio, Tráfico, Fraude, Outros
- Tipo de Comunicação: Boletim de Ocorrência, Denúncia Anônima, Relatório de Investigação, Ofício, Canal de Ética, Outro
- Unidade: Unidade Central, Unidade Norte, Unidade Sul, Unidade Leste, Unidade Oeste
- Gravidade: Baixa, Média, Alta, Crítica
- Latitude: Coordenada de latitude (ex: -23.5505)
- Longitude: Coordenada de longitude (ex: -46.6333)
- Data Início: Data de início da ocorrência (formato: YYYY-MM-DD)
- Data Fim: Data de fim da ocorrência (opcional, formato: YYYY-MM-DD)
- Responsável: Nome do responsável pela ocorrência
- Status: Em andamento, Finalizada
- Observações: Observações sobre a ocorrência
- Considerações Finais: Considerações finais sobre a ocorrência (opcional)

INSTRUÇÕES:
1. Abra este arquivo Excel (.xlsx) no Microsoft Excel ou Google Sheets
2. Preencha os dados seguindo o formato das linhas de exemplo
3. Salve o arquivo como Excel (.xlsx) ou CSV
4. Use o botão "Upload Excel" no sistema para importar os dados

OBSERVAÇÕES:
- Não altere os nomes das colunas
- Mantenha o formato das datas como YYYY-MM-DD
- As coordenadas devem estar no formato decimal (ex: -23.5505, -46.6333)
- Campos obrigatórios: Tipo de Ocorrência, Tipo de Comunicação, Gravidade, Data Início, Responsável, Status
- Este arquivo Excel já contém formatação e validações adequadas
